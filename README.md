# Memory Card Game

A memory card game built with React as part of The Odin Project curriculum.  
The game uses the Rick and Morty API to fetch character images and names and challenges players to click each card only once.

## Built With

- React
- Vite
- Tailwind CSS
- Rick and Morty API

## Features

- Fetches character data from an external API
- Displays character cards with images and names
- Cards shuffle after every click
- Score increases when a card is clicked for the first time
- Game resets when the same card is clicked twice

## How to Play

1. Click on a card to gain a point.
2. Each card can only be clicked once.
3. After each click, the cards shuffle.
4. Clicking the same card twice resets the score.
5. Try to get the highest score possible.

## Installation

Clone the repository:

```bash
git clone https://github.com/Cage4D/Memory-Card.git
cd Memory-Card
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Rick and Morty API
